Antes de React

El objetivo de esta tarea es crear una página HTML con estructura semántica correcta y estilos básicos CSS, aplicando etiquetas, selectores, variables y buenas prácticas.

Navegación

  • HTML
  • CSS
  • JavaScript

HTML

El HTML (HyperText Markup Language) es el lenguaje de marcado que se usa para estructurar el contenido de una página web. Define elementos como títulos, párrafos, listas, imágenes, formularios y enlaces, formando la base de cualquier sitio.

CSS

El CSS (Cascading Style Sheets) se encarga de dar estilo y diseño al HTML. Permite definir colores, tipografías, márgenes, posiciones y adaptabilidad, separando la estructura del contenido de la presentación visual.

JavaScript

JavaScript es el lenguaje de programación que aporta interactividad y dinamismo a las páginas web. Con él se pueden validar formularios, crear animaciones, actualizar datos sin recargar la página y desarrollar aplicaciones completas en el navegador.
